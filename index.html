Distances to Lines v2.2

Proof app: warm up GPS, start a run with a 30 second settle countdown, detect fixed trip-line crossings, and capture the driven distance at each line. Distance-to-next is display-only. Line crossing remains the route trigger; waypoint/radius distance is recorded as a passive comparison.
Test route: Bluefin line → Noordhoek Sign → Ruby Way → 90 Sign.

1. GPS / Run Control

GPS starts when the app opens. Wait for a stable GPS status, press Start Run before Bluefin, then allow the 30 second run countdown before crossing the first line.
